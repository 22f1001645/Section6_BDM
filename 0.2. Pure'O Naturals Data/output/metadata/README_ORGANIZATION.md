Metadata Output Organization

Subfolders:
- package: final deliverables and bundled package
- data: computed CSV/TXT outputs used by reports
- reports: validation and reconciliation reports
- docs: documentation and outlines
- logs: operational logs from reconciliation processes
- tmp: temporary verification working directories
- archives: backups and older artifacts (pre-reorg snapshot: metadata_pre_reorg_20251107_225611.zip)

Latest refinement package is stored at package/refinement_latest.zip.
